Introduction to 1D Arrays

What is an Array?

In Python, we use lists as arrays. A list is an ordered collection of items that can store multiple values.

# Creating a 1D array (list) numbers = [10, 20, 30, 40, 50] names = ["Alice", "Bob", "Charlie"] mixed = [1, "hello", 3.14, True]

Accessing Elements

We use index numbers to access elements. Remember: Python uses 0-based indexing!

fruits = ["apple", "banana", "cherry", "date"] print(fruits[0]) # First element: apple print(fruits[2]) # Third element: cherry print(fruits[-1]) # Last element: date print(fruits[-2]) # Second last: cherry

Common Mistake Alert! 🚨

Index Out of Range Error
If you have a list with 5 elements, valid indices are 0-4.
Trying to access index 5 will cause an error!

Manipulating 1D Arrays

Adding Elements

# Adding elements shopping = ["milk", "bread"] shopping.append("eggs") # Adds to end: ["milk", "bread", "eggs"] shopping.insert(1, "butter") # Insert at position 1: ["milk", "butter", "bread", "eggs"]

Removing Elements

shopping.remove("butter") # Removes first occurrence last_item = shopping.pop() # Removes and returns last item shopping.pop(0) # Removes item at index 0

A Note on the OCR Specification

While methods like .insert(), .remove(), and .pop() are extremely useful for general programming, projects, and competitions, it's important to know that for the OCR J277 specification, you are primarily expected to know and use .append() for adding items to a list.

Learning these additional methods will make you a stronger programmer, but they are considered outside the core specification and are not directly examinable.

Introduction to 2D Arrays

What is a 2D Array?

A 2D array is like a table or grid - it's a list that contains other lists!

# Creating a 2D array (3x3 grid) grid = [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ] # Accessing elements: grid[row][column] print(grid[0][0]) # Top-left: 1 print(grid[1][1]) # Center: 5 print(grid[2][2]) # Bottom-right: 9

Student test scores can be stored in a 2D array:

/ 5

(a) 3 rows and 4 columns [1 mark]

(b) seats[2][3] = 1 [2 marks - 1 for correct indices, 1 for assignment]

count = 0

    for seat in row:

            count += 1

Key Takeaways 🎯

  • 1D arrays (lists) store data in a single row
  • Access elements using index notation: array[index]
  • Python uses 0-based indexing (first element is at index 0)
  • 2D arrays are lists of lists - like tables or grids
  • Access 2D array elements with: array[row][column]
  • Arrays are used everywhere: games, images, data storage
  • Common methods: append(), insert(), remove(), pop()

Common Mistakes to Avoid ⚠️

  • Index out of range: Remember arrays start at 0, not 1!
  • Confusing rows and columns: In array[row][column], row comes first
  • Forgetting brackets: Use array[0][0], not array[0,0]
  • Modifying while iterating: Be careful when removing items in a loop
