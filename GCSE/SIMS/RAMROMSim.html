BIOS v2.4

⚡Step 1: Power Button Pressed

You switch on the computer, sending power to all components.

🔍Step 2: BIOS/UEFI Checks

The BIOS runs a POST (Power-On Self-Test) to check essential hardware like CPU, RAM, and storage devices.

💾Step 3: Find Operating System

BIOS searches for the operating system on your hard drive or SSD where it's permanently stored.

📥Step 4: Load OS into Memory

The operating system's core components are loaded from secondary storage into RAM for fast access.

✅Step 5: Ready to Use

Boot complete! Your login screen or desktop appears, and the computer is ready for use.

Understanding ROM (Read-Only Memory)

Key Features of ROM:

  • • Non-volatile - keeps data without power
  • • Read-only - cannot be overwritten
  • • Stores BIOS/boot instructions
  • • Essential for starting the computer

Virtual Memory: When RAM Runs Out

What is Virtual Memory?

When RAM is full, the computer uses part of the hard disk as temporary RAM. This is called virtual memory.

⚠️ Virtual memory is much slower than real RAM!
