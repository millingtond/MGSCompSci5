Understanding Bit Limitations

See how the number of bits affects how many characters we can represent

With 8 bits, you can represent:

256 different characters

Formula: 2^8 = 256

📊 Common Character Sets:

  • • 7 bits = 128 characters (Original ASCII)
  • • 8 bits = 256 characters (Extended ASCII)
  • • 16 bits = 65,536 characters (Basic Unicode)
  • • 32 bits = 4,294,967,296 characters (Full Unicode)

ASCII vs Unicode

Understanding different character encoding standards

ASCII

  • ✓ 8 bits per character
  • ✓ 256 possible characters
  • ✓ English letters, numbers, symbols
  • ✗ No emojis or special characters
  • ✗ Limited language support

Unicode

  • ✓ 16-32 bits per character
  • ✓ Over 1 million possible characters
  • ✓ All languages supported
  • ✓ Emojis and special symbols
  • ✓ Global standard

Musical Binary Representation:

🎼 How It Works

Each bit in the binary representation triggers a different note:

  • • Bit 1 (rightmost) = C note
  • • Bit 2 = D note
  • • Bit 3 = E note
  • • Bit 4 = F note
  • • Bit 5 = G note
  • • Bit 6 = A note
  • • Bit 7 = B note
  • • Bit 8 = High C note
