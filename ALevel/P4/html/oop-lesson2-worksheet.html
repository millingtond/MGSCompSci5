📊 Attributes (Properties)

Attributes are the data members of a class - they store information about the object.

Understanding Attributes

What are Attributes?

Think of attributes as the characteristics of an object:

  • A Person has: name, age, height
  • A Car has: color, model, speed
  • A Game Character has: health, level, inventory
Data Types

Attributes have specific data types:

  • STRING: name, color, description
  • INTEGER: age, level, count
  • REAL: height, weight, price
  • BOOLEAN: isAlive, hasKey

⚡ Methods (Behaviors)

Methods are functions that belong to a class and define what objects can do!

Pseudocode Methods

CLASS Character
    // ... attributes ...

    PUBLIC PROCEDURE Attack()
        // Attack code here
    ENDPROCEDURE

    PUBLIC FUNCTION GetHealth() RETURNS INTEGER
        RETURN Health
    ENDFUNCTION
ENDCLASS

Python Methods

class Character:
    # ... attributes ...

    def attack(self):
        # Attack code here
        pass

    def get_health(self):
        return self.__health

Method Types

🔧 Procedures

Perform actions but don't return values

Examples:

  • move()
  • attack()
  • print_status()
📤 Functions

Perform actions AND return values

Examples:

  • get_health()
  • calculate_damage()
  • is_alive()

🔨 Constructors

Constructors are special methods that run automatically when an object is created!

Constructor Demonstration

🎯 Purpose of Constructors

Constructors are used to:

  • ✅ Initialize object attributes with starting values
  • ✅ Set up the object's initial state
  • ✅ Perform any setup tasks needed
  • ✅ Ensure objects start in a valid state

Key Point: Constructors run automatically - you don't call them directly!

📝 Constructor Syntax
Pseudocode
PUBLIC PROCEDURE New(parameters)
    // Initialization code
ENDPROCEDURE
Python
def __init__(self, parameters):
    # Initialization code
📥 Constructor Parameters
class Player:
    def __init__(self, name, health, level):
        self.__name = name
        self.__health = health
        self.__level = level

# Creating an object passes values to constructor
player1 = Player("Arion", 100, 1)
🎮 Full Example
class Enemy:
    def __init__(self, enemy_type, damage):
        # Set attributes based on parameters
        self.__type = enemy_type
        self.__damage = damage
        
        # Set default values
        self.__health = 50
        self.__is_alive = True

# Create different enemies
goblin = Enemy("Goblin", 10)
dragon = Enemy("Dragon", 50)

✨ Creating Objects (Instantiation)

Instantiation is the process of creating objects from a class!

Pseudocode Instantiation

// Declare object variable
DECLARE player1 : Character

// Create object instance
player1 ← NEW Character("Hero", 100)

Python Instantiation

# Create object instance
player1 = Character("Hero", 100)

# Create multiple objects
player2 = Character("Mage", 80)
enemy1 = Enemy("Goblin", 50)

🎮 Building RPG Classes

Now let's apply everything we've learned to build our RPG classes!

Complete RPG Class Implementation

Let's build a complete Player class for our game:

class Player:
    def __init__(self, name, character_class):
        # Initialize attributes
        self.__name = name
        self.__class = character_class
        self.__level = 1
        self.__health = 100
        self.__max_health = 100
        self.__experience = 0
        self.__gold = 0

    def attack(self, target):
        damage = 10 * self.__level
        print(f"{self.__name} attacks for {damage} damage!")
        return damage

    def heal(self, amount):
        self.__health = min(self.__health + amount, self.__max_health)
        print(f"{self.__name} healed for {amount} HP!")

    def level_up(self):
        self.__level += 1
        self.__max_health += 20
        self.__health = self.__max_health
        print(f"{self.__name} reached level {self.__level}!")

📝 Summary & Final Quiz

Key Takeaways

  • Classes: Blueprints or templates for creating objects
  • Objects: Instances of classes with specific values
  • Attributes: Data members that store object information
  • Methods: Functions that define object behaviors
  • Constructors: Special methods that initialize objects
  • Instantiation: The process of creating objects from classes
