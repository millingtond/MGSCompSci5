üìù Summary & Final Quiz

Key Takeaways

  • Sampling converts continuous analogue waves to discrete digital values
  • Sample resolution (bit depth) determines the number of distinct values (2^n)
  • Sampling rate is the frequency of samples taken (Hz)
  • Higher resolution/rate = better quality but larger files
  • CD standard uses 16-bit/44.1kHz as optimal compromise
  • File size = rate √ó resolution √ó time √ó channels
