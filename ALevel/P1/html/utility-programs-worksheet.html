  • Storing backups in the same location as originals
  • Not encrypting sensitive backup data
  • Irregular backup schedules
  • Not verifying backup integrity

Explain two functions of disk repair software and describe how it handles bad sectors.

Mark Scheme:

Award 1 mark for each valid function (max 2) and 1 mark for each explanation:

  • Checks for errors/inconsistencies on disk (1)
  • Retrieves files from damaged disk / reconstructs directory (1)
  • Marks bad sectors as unusable (1) - prevents data being written to damaged areas (1)

A company is implementing a new security system. Describe three ways utility programs can improve system security.

Mark Scheme:

Award 2 marks per method (1 for naming, 1 for description):

  • Firewall (1) - monitors/blocks unauthorized network access (1)
  • Anti-virus (1) - detects and removes malicious software (1)
  • Authentication software (1) - verifies user identity to prevent unauthorized access (1)
  • Backup software (1) - protects against data loss from attacks (1)
  • Automatic updates (1) - ensures latest security patches installed (1)
