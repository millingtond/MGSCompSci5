Understanding Storage Sizes

There are two systems for measuring data storage: Binary (IEC) and Decimal (SI)

Unit Binary (IEC) Decimal (SI) Difference
Kilobyte/Kibibyte 1 KiB = 1,024 bytes 1 KB = 1,000 bytes 2.4%
Megabyte/Mebibyte 1 MiB = 1,048,576 bytes 1 MB = 1,000,000 bytes 4.9%
Gigabyte/Gibibyte 1 GiB = 1,073,741,824 bytes 1 GB = 1,000,000,000 bytes 7.4%
Terabyte/Tebibyte 1 TiB = 1,099,511,627,776 bytes 1 TB = 1,000,000,000,000 bytes 10.0%

üìù Summary & Final Quiz

Key Takeaways

  • Binary is base 2, using only 0 and 1
  • Binary to Denary: Add place values where bit = 1
  • Denary to Binary: Repeatedly subtract largest power of 2
  • BCD represents each decimal digit as 4 bits
  • Hexadecimal is base 16, using 0-9 and A-F
  • One hex digit represents 4 binary bits
  • Binary prefixes (KiB, MiB) use powers of 1024
  • Decimal prefixes (KB, MB) use powers of 1000
