üìã Exam Practice Questions

Question 1 (6 marks)

Describe how the browser uses the Domain Name Service (DNS) to display a web page when a user enters a URL.

Mark Scheme:
  • URL is passed to the nearest Domain Name Server (1 mark)
  • DNS server stores a database/list of URLs and matching IP addresses (1 mark)
  • DNS (Name Resolver) looks for the URL in its database (1 mark)
  • Finds the matching IP address and returns it (1 mark)
  • Or if it cannot find it, forwards to another DNS server at higher level (1 mark)
  • DNS server adds returned IP address to cache and returns to browser (1 mark)

Question 2 (4 marks)

Explain two benefits of DNS caching in the context of internet performance.

Mark Scheme (2 marks per benefit):
  • Reduces repeated DNS lookups (1) saving bandwidth/network traffic (1)
  • Faster page loading (1) as IP already stored locally (1)
  • Reduces load on DNS servers (1) improving overall system performance (1)
  • Provides redundancy (1) if DNS server temporarily unavailable (1)

üìù Summary & Final Quiz

Key Takeaways

  • DNS translates domain names to IP addresses
  • DNS servers store databases of domain-IP mappings
  • Resolution process may involve multiple DNS servers
  • Caching improves performance and reduces load
  • Hierarchy distributes the DNS workload globally
  • URLs contain multiple components, DNS resolves the domain
