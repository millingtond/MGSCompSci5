Practice Exam Questions

Test your knowledge with these exam-style questions. Enter your predicted marks before viewing the mark scheme.

Question 1 (6 marks)

Explain the difference between security and integrity in the context of data protection. Give one example of a method used to ensure each.

Mark Scheme:

  • Security is protecting data from loss/damage (1 mark)
  • Integrity is ensuring data is accurate/correct/valid (1 mark)
  • Security example: passwords/firewalls/access rights (1 mark)
  • Integrity example: validation/verification/parity checks/checksums (1 mark)
  • Clear explanation of the difference (1 mark)
  • Appropriate examples that clearly relate to each concept (1 mark)

Question 2 (8 marks)

A company has been infected with a worm. Describe what a worm is and explain how it differs from a virus. Suggest two ways the company could have protected against this attack.

Mark Scheme:

  • Worm is standalone malware (1 mark)
  • Does not require user interaction to spread (1 mark)
  • Exploits vulnerabilities automatically (1 mark)
  • Virus requires user action/attached to files (1 mark)
  • Virus replicates by copying itself (1 mark)
  • Protection method 1: Keep software updated/patch vulnerabilities (1 mark)
  • Protection method 2: Use firewall/antivirus/network monitoring (1 mark)
  • Clear comparison between worm and virus (1 mark)
