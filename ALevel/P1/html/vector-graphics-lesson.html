üîß Vector Graphics Components

Key Definitions

1. Vector Graphics

An image made up of drawing objects where the properties of each object determine its shape and appearance.

2. Drawing List

A set of commands used to define a vector image. It contains the commands required to draw each object within the image.

3. Drawing Object

An individual object being drawn within a vector image (e.g., line, text, rectangle, circle).

4. Properties

The values required to create the object (e.g., x/y coordinates, width, height, fill color, stroke color).

Build a Vector Shape

Create a rectangle by setting its properties:
