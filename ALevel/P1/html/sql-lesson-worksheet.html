🔧 SQL Fundamentals

What is SQL?

SQL (Structured Query Language) is a standardized language for managing and manipulating relational databases. It's provided by a Database Management System (DBMS) to support all database operations.

Interactive: DDL vs DML

SQL is divided into two main categories. Click on each to learn more:

DDL - Data Definition Language

Creates or alters table structures

Examples: CREATE, ALTER, DROP

Only creates structure, no data

DML - Data Manipulation Language

Manages data within tables

Examples: SELECT, INSERT, UPDATE, DELETE

Works with actual data
