üîÄ Router: The Network Director

üîÄ
Router

Connects two or more networks and forwards data packets between them

Key Functions

  • Connects two (or more) networks
  • Can act as a hardware firewall
  • Forwards data packets using routing tables
  • Each forwarding is called a "hop"
  • Provides public/routable IP addresses
  • Performs NAT (Network Address Translation)

üîå Switch: The Smart Connector

üîå
Switch

Central device connecting network devices using MAC addresses

Switch Advantages

  • Traffic directed only to intended recipients (unicast/multicast)
  • No collisions possible (full duplex connections)
  • Uses MAC address table for routing decisions
  • More secure than hubs
  • Less bandwidth wastage

üåÅ Bridge: The LAN Connector

üåÅ
Bridge

Connects LANs that use the same protocols

Bridge Functions

  • Connects one LAN to another LAN
  • Both networks must use same communication protocols
  • Can connect parts of a LAN to form one larger network
  • Uses MAC address table for routing decisions
  • Forwards data packets to next part of network

WAP Functions

  • Connects wireless devices to wired LAN

üí≥ Network Interface Cards (NIC/WNIC)

üîå
NIC

Wired network interface

üì°
WNIC

Wireless network interface

NIC Functions

  • Provides device with unique MAC address
  • Hardware interface between computer and network
  • Physical connection to Ethernet cable (NIC)
  • Converts between digital and analogue signals (WNIC)
  • Encrypts/decrypts data
  • Checks incoming transmissions for correct MAC address

üìù Summary & Final Quiz

Key Takeaways

  • Router: Connects networks, uses routing tables, performs NAT
  • Gateway: Bridges networks with different protocols
  • Switch: Connects devices using MAC addresses, no collisions
  • Bridge: Connects LANs with same protocols
  • Modem: Converts between digital and analogue signals
  • WAP: Provides wireless access to wired networks
  • NIC/WNIC: Provides MAC address and network interface
