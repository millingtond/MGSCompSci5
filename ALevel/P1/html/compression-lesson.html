What is Compression?

Compression is a method or protocol for using fewer bits to represent the original information. It's like packing a suitcase efficiently - you fit the same items into less space!

‚öñÔ∏è Lossy vs Lossless Compression

Key Differences

Lossy Compression

  • Removes "useless" information permanently
  • Cannot recover original data
  • Much smaller file sizes (can reduce to ~10%)
  • Some quality loss
  • Examples: JPEG, MP3, MPEG

Lossless Compression

  • Every bit can be recovered
  • Perfect reconstruction of original
  • Larger file sizes (max ~50% reduction)
  • No quality loss
  • Examples: ZIP, PNG, FLAC

Lossy Audio Compression (MP3)

MP3 uses psycho-acoustic modeling to remove sounds we can't hear:

How MP3 Works

  1. Uses psycho-acoustic modeling
  2. Applies perceptual music/noise shaping
  3. Removes sounds the human ear can't hear
  4. Keeps sounds we hear better than others
  5. If two sounds play together, discards the softer one
