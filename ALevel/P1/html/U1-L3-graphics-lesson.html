What is a Pixel?

A pixel (picture element) is the smallest identifiable area of a digital image. Each pixel:

  • Is a single color
  • Has a binary value representing that color
  • Can be changed by modifying its binary value

üé® Color Depth & Bit Depth

Understanding Color Depth

Color depth (or bit depth) defines how many colors each pixel can represent:

  • Each pixel can be n bits
  • Number of possible colors = 2n
  • Higher bit depth = greater color range = better quality
  • Monochrome: Different shades of black, white, and grey only

üìã Image Metadata

What is Metadata?

Metadata is data that describes other data. For images, this includes essential information about the file.

Common Image Metadata

  • File type (e.g., JPEG, PNG, BMP)
  • File size
  • Location/offset of image data within the file
  • Dimensions in pixels
  • Color depth (bits per pixel)
  • Type of compression used, if any
