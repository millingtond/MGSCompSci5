üîß Bit Streaming Fundamentals

What is Bit Streaming?

Bit streaming is a method of transmitting data continuously as a series of bits, allowing users to access content without downloading entire files first.

Fill in the Blanks

Complete the sentences by selecting the correct terms:

1. Data is before transmitting to reduce file size.

Key Components

  • Compression: Reduces data size for efficient transmission
  • Media Server: Hosts and distributes the streaming content
  • Buffer: Temporary storage for smooth playback
  • Client Software: Receives and plays the bit stream

Real-time Streaming Process

  1. Event captured live with video camera
  2. Camera connected to computer
  3. Video signal converted to streaming format (encoded)
  4. Encoded feed uploaded to streaming server
  5. Server sends live images to all requesting users

üìù Exam Practice Questions

Question 1 (4 marks)

Describe two benefits and two drawbacks of using bit streaming to watch films online.

Mark Scheme:

Benefits (1 mark each, max 2):

  • No need to wait for entire file to download
  • No need to store large files on user's computer
  • Allows on-demand playback
  • No specialist software required for browser playback

Drawbacks (1 mark each, max 2):

  • Video stops/hangs if internet speed is slow
  • Video stops if inadequate buffering capacity
  • Loss of internet means no access to films
  • May require specific software to run files
  • Viruses can be downloaded from websites

Question 2 (6 marks)

Explain the differences between real-time and on-demand bit streaming. Include the process for each type.

Mark Scheme:

Differences (3 marks):

  • Real-time: live stream of current event / On-demand: pre-recorded content (1)
  • Real-time: cannot pause/rewind / On-demand: full playback control (1)
  • Real-time: captured with live camera / On-demand: existing files encoded (1)

Process explanation (3 marks):

Key Takeaways

  • Buffering ensures smooth playback by storing data temporarily
  • On-demand allows full control, real-time is live only
  • High-speed internet is essential for quality streaming
