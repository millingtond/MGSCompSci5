📊 K-Map Fundamentals

What are Karnaugh Maps?

Karnaugh Maps (K-Maps) are a graphical method for simplifying Boolean expressions. They provide a visual way to identify patterns and minimize logic circuits.

📏 The 8 Grouping Rules

Interactive Rule Explorer

Click on each rule to see an example:

Rule 1: Only 1s

Groups must only contain cells with value 1

Rule 2: No Diagonals

Groups must be horizontal or vertical

Rule 3: Powers of 2

Groups must contain 2^n cells (1,2,4,8...)

Rule 4: Maximize Groups

Each group should be as large as possible

Rule 5: Overlapping Allowed

Groups may overlap each other

Rule 6: Wrapping Allowed

Groups may wrap around edges

Rule 7: Cover All 1s

Every 1 must be in at least one group

Rule 8: Minimize Groups

Use as few groups as possible

📝 Summary & Final Quiz

Key Takeaways

  • K-maps provide a visual method for Boolean simplification
  • Groups must follow all 8 rules to be valid
  • Expressions are derived from constant variables in groups
  • K-maps wrap like a torus - edges connect
  • Optimal grouping minimizes the final expression

📋 Exam Practice Questions

Question 1 (6 marks)

A student is designing a circuit with 3 inputs (A, B, C). The output is 1 when:

  • All inputs are 0
  • A and B are 1, C is 0
  • A is 0, B and C are 1

a) Draw the K-map (3 marks)
b) Find the minimal expression (3 marks)

Mark Scheme:

Part a) K-map (3 marks):

  • Correct K-map structure with labels (1 mark)
  • Correct placement of 1s: positions 000, 110, 011 (2 marks)

Part b) Expression (3 marks):

  • Identifying valid groups (1 mark)
  • Correct terms from groups (1 mark)
  • Final expression: ĀB̄C̄ + BC̄ + ĀBC or equivalent (1 mark)

Question 2 (4 marks)

Explain two advantages of using Karnaugh Maps over Boolean algebra for logic simplification.

Mark Scheme:

Award 2 marks per advantage (1 for stating, 1 for explanation):

  • Visual method - easier to spot patterns
  • Systematic approach - reduces errors
  • Handles don't care conditions easily
  • No need to remember Boolean laws
  • Faster for small number of variables
