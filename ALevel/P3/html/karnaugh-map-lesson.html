What are Karnaugh Maps?

Karnaugh Maps (K-Maps) are a graphical method for simplifying Boolean expressions. They provide a visual way to identify patterns and minimize logic circuits.

üìè The 8 Grouping Rules

Interactive Rule Explorer

Click on each rule to see an example:

Rule 1: Only 1s

Groups must only contain cells with value 1

Rule 2: No Diagonals

Groups must be horizontal or vertical

Rule 3: Powers of 2

Groups must contain 2^n cells (1,2,4,8...)

Rule 4: Maximize Groups

Each group should be as large as possible

Rule 5: Overlapping Allowed

Groups may overlap each other

Rule 6: Wrapping Allowed

Groups may wrap around edges

Rule 7: Cover All 1s

Every 1 must be in at least one group

Rule 8: Minimize Groups

Use as few groups as possible

Key Takeaways

  • K-maps provide a visual method for Boolean simplification
  • Groups must follow all 8 rules to be valid
  • Expressions are derived from constant variables in groups
  • K-maps wrap like a torus - edges connect
  • Optimal grouping minimizes the final expression
