Key Takeaways

üìã Exam Practice Questions

Question 1 (6 marks)

Explain three advantages of packet switching over circuit switching for internet data transmission.

Mark Scheme:

Award 2 marks per advantage (1 for naming, 1 for explanation):

  • Efficient bandwidth usage - multiple users can share the same network paths
  • Resilience - if one route fails, packets can take alternative routes
  • Cost-effective - no need to reserve entire path for duration
  • Scalability - can handle varying traffic loads
  • Error handling - only corrupted packets need retransmission

Question 2 (4 marks)

Describe how a router uses its routing table to forward packets. Include the role of metrics in your answer.

Mark Scheme:
  • Router examines destination IP in packet header (1)
  • Looks up destination in routing table (1)
  • Identifies possible routes/next hops (1)
  • Uses metrics to select route with lowest cost (1)
