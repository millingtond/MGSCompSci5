👨‍🏫 Supervised Learning

Supervised Learning Process

# Supervised Learning Steps:
1. Input + Expected Output → Training
2. Model learns patterns from labelled data
3. Test with new labelled data
4. Compare results with expected output
5. Refine if errors found
6. Use with unlabelled data for predictions

📝 Summary & Final Quiz

Key Takeaways

  • Machine Learning extracts knowledge from data using algorithms
  • Three types of AI: Narrow (specific tasks), General (human-like), Strong (superhuman)
  • Supervised Learning uses labelled data for predictions
  • Unsupervised Learning finds patterns in unlabelled data
  • Reinforcement Learning learns through trial and error with rewards

Question 1 (4 marks)

Define machine learning and explain how it differs from traditional programming. Include two examples of machine learning applications.

Mark Scheme:
  • Definition: Extraction of knowledge from data using algorithms (1 mark)
  • Difference: ML learns from data vs traditional programming with fixed rules (1 mark)
  • Example 1: Any valid ML application (1 mark)
  • Example 2: Different valid ML application (1 mark)

Question 2 (6 marks)

Compare and contrast supervised, unsupervised, and reinforcement learning. For each type, explain:

  • The type of data used
  • How the learning process works
  • A suitable application
Mark Scheme:

Award 2 marks per learning type (0.5 for data type, 1 for process, 0.5 for application):

  • Supervised: Labelled data, learns from examples with known outputs, e.g., spam filtering
  • Unsupervised: Unlabelled data, discovers patterns automatically, e.g., customer segmentation
  • Reinforcement: No training data, learns through trial and error with rewards, e.g., game AI
