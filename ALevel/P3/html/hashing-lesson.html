What is Hashing?

A hashing algorithm is a mathematical formula used to perform a calculation on input data. It's commonly used in direct access methods on random and sequential files.

üí• Hash Collisions

A hash collision occurs when the hashing algorithm produces the same storage location for more than one record. This is a common challenge that needs to be resolved.

What is a Digital Signature?

A digital signature is a means of electronically signing a document with data that cannot be forged. It serves two main purposes:

  • To ensure a document is authentic and came from a trusted source
  • To ensure a document has not been altered during transmission

Key Takeaways

  • Hashing creates fixed-size outputs from variable inputs
  • Hash collisions can be resolved using linear progression or overflow areas
  • Software authentication uses hashing and encryption to verify authenticity
  • Digital signatures ensure documents are authentic and unaltered
  • Certificate Authorities are trusted third parties that issue certificates
  • Digital certificates contain public keys and identifying information
