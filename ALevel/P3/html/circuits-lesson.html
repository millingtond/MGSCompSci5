Limitation of Half Adder

The half adder can only add two single bits. It cannot accept a carry input from a previous addition, limiting its use in multi-bit arithmetic.

Key Point: OR vs XOR

The final gate before Cout can be either OR or XOR without changing the logic. This is because the two carry outputs can never both be 1 simultaneously in a full adder.

Application Summary

  • Half/Full Adders: Arithmetic Logic Units (ALUs), calculators
  • SR Latches: Basic memory cells, switch debouncing
  • JK Flip Flops: Counters, frequency dividers, shift registers
  • General: Computer memory, digital clocks, state machines

üìù Summary & Final Quiz

Key Takeaways

  • Combinational circuits output depends only on current inputs
  • Sequential circuits have memory and depend on previous states
  • Half adders add two bits: S = A ‚äï B, C = A ¬∑ B
  • Full adders add three bits using two half adders
  • SR latches store one bit but have an invalid state
  • JK flip flops are clocked and eliminate the invalid state
