üìù Summary & Final Quiz

Key Takeaways

  • Protocols provide standards for communication between different devices
  • TCP/IP has 4 layers: Application, Transport, Internet/Network, and Link
  • Each layer has specific responsibilities and can only communicate with adjacent layers
  • Different protocols serve different purposes (HTTP for web, SMTP for email, etc.)
  • Layering provides modularity, specialization, and interoperability benefits

üìã Exam Practice Questions

Question 1 (6 marks)

Explain the four layers of the TCP/IP protocol stack and describe one function of each layer.

Mark Scheme:
  • Application Layer - 1 mark for naming, 1 mark for function (e.g., determines protocol to use)
  • Transport Layer - 1 mark for naming, 1 mark for function (e.g., manages reliable delivery)
  • Internet Layer - 1 mark for naming, 1 mark for function (e.g., routes packets)
  • Link Layer - 1 mark for naming, 1 mark for function (e.g., handles physical transmission)

Maximum 1 mark per layer for function, must be specific and accurate.

Question 2 (4 marks)

Describe two advantages of using a layered protocol architecture.

Mark Scheme:

2 marks per advantage (1 for identifying, 1 for explanation):

  • Modularity - layers can be changed without affecting others
  • Specialization - developers can focus on one layer
  • Interoperability - different vendors can create compatible products
  • Easier debugging - problems can be isolated to specific layers
