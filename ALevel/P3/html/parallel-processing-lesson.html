📝 Summary & Final Assessment

Key Takeaways

  • Three Levels: Instruction (pipelining), Processor (multi-core), Computer (networked)
  • Four Systems: SISD, SIMD, MISD, MIMD - each for different use cases
  • Multi-programming: Scheduling allows apparent simultaneous execution
  • MPCs: Massive parallelism requires careful software and hardware design
  • Real Impact: From weather prediction to gaming, parallel processing is everywhere

📋 Exam Practice Questions

Question 1 (6 marks)

Explain the three levels of parallelism in computer systems. For each level:

  • Name the level (1 mark each)
  • Describe how it achieves parallelism (1 mark each)
Mark Scheme:
  • Instruction-level (1) - Uses pipelining to work on multiple instructions (1)
  • Processor-level (1) - Multiple cores/processors work on different parts (1)
  • Computer-level (1) - Networked computers work together as a unit (1)

Question 2 (8 marks)

A company wants to process large datasets for climate modeling.

(a) Recommend which parallel system type (SISD/SIMD/MISD/MIMD) would be most suitable and justify your choice. (4 marks)

(b) Describe two software considerations and two hardware considerations for implementing this system. (4 marks)

Mark Scheme:

(a) MIMD recommended (1) because:

  • Different regions can be processed with different models (1)
  • Asynchronous operation allows flexibility (1)
  • Can handle complex, varied calculations (1)

(b) Software: Code must be split into parallel blocks (1), Requires coordination between processors (1)

Hardware: Need communication links between processors (1), Requires message-passing infrastructure (1)
