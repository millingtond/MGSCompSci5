Anti-malware Software

Anti-malware (often including Antivirus) software scans files for viruses. It detects/identifies a virus by comparing data to a database of known viruses (virus definitions/signatures). It can run in the background and get automatic updates. If a virus is found, it alerts the user and can quarantine or delete the virus. It also helps stop the download of spyware/malware.

Firewall

A Firewall examines incoming and outgoing network traffic. Based on a set of rules/criteria, it blocks/filters traffic that doesn't meet these rules. This helps in blocking unauthorized access and malicious traffic. It can also block access to specified IP addresses/websites and warn users of attempted unauthorized access by software on their computer.

1. Describe two functions of anti-malware software. [2 marks]

Mark Scheme:

  • Scans files/programs for known malware signatures (1 mark).
  • Detects/identifies malware (1 mark).
  • Prevents malware from running/installing (1 mark).
  • Removes/deletes malware from the system (1 mark).
  • Quarantines suspicious files (1 mark).
  • Uses heuristic analysis to detect new/unknown malware (1 mark).
  • (Award 1 mark for each distinct correct function, max 2 marks)

2. Explain how a firewall helps to protect a computer or network. [3 marks]

Mark Scheme:

  • Monitors incoming and/or outgoing network traffic (1 mark).
  • Blocks traffic based on a set of predefined rules or criteria (1 mark).
  • Acts as a barrier between a trusted network (e.g. LAN) and an untrusted network (e.g. Internet) (1 mark).
  • Can block access to/from specific IP addresses or ports (1 mark).
  • Helps prevent unauthorized access to the system/network (1 mark).
  • (Award 1 mark for each distinct correct point, max 3 marks)

3. What is the main purpose of penetration testing? [2 marks]

Mark Scheme:

  • To identify security vulnerabilities/weaknesses in a system or network (1 mark).
  • By simulating an attack / performing authorized hacking to see if security can be breached (1 mark).

Key Takeaways

  • Anti-malware software detects, prevents, and removes various types of malware using signature databases and heuristic analysis.
  • Firewalls monitor and control network traffic based on rules, acting as a barrier between trusted and untrusted networks.
  • Penetration Testing is an authorized simulated attack to find and fix security vulnerabilities before malicious attackers can exploit them.
